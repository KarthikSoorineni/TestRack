# TestRack


Steps to Run the application
1) Download the above TestRack.zip file and unzip it in the local machine.
2) Open the build.gradle file in this folder as a package in Intellij or any of the IDEs.
3) When the project structure is imported, then run the DemoApplication.java which is under src/main/java/com/example.
4) hit the api http://localhost:8082/api/Hello%20World.
